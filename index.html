Lie Group Integrator Double Spherical Pendulum Documentation¶

lie_group_integrator_double_spherical_pendulum is a MATLAB code where different Lie group integrators are tested on the example of the double spherical pendulum. The code is part of the source code developed at the Department of Mathematical Sciences at NTNU, for the paper (Celledoni, Çokaj, Leone, Murari and Owren, (2021) International Journal of Computer Mathematics).
